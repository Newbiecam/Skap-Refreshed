## ✨ Overview

**Skap | Refreshed** is a redesigned and expanded version of the original Skap experience, featuring:

- ⚡ Smooth physics & responsive movement  
- 🧱 Built-in map editor  
- 🎮 Custom abilities / powers system  
- 💬 Integrated chat & player system  
- 🎨 Fully customizable UI & themes  

---

## 🚀 Features

### 🧠 Core Gameplay
- Fast-paced movement system  
- Momentum-based physics  
- Dynamic player interactions  

### 🛠️ Map Editor
- Create maps in real-time  
- Save / load / export JSON  
- Grid snapping + precision tools  

### ⚡ Powers System
- Equip and switch abilities  
- Cooldowns + visual feedback  
- Passive & active mechanics  

### 💬 Social System
- Global / Party / Whisper chat  
- Multi-player simulation tools  
- Account system (local session)  

### 🎨 UI & Customization
- Glass-style HUD  
- Theme colors (background + walls)  
- Toggleable overlays (FPS, hitbox, etc.)
